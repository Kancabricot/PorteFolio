Listrik is my very first video game project, which I developed entirely on my own. It gave me the opportunity to explore and experiment with all the stages of video game production, from start to finish.

This is a single-player puzzle-platformer where the goal is to solve levels designed around a battery mechanic carried by a small robot named Listrik.

The game was designed to run as a web page using JavaScript and the Phaser 3 framework.

Player


You play as a small robot named Listrik who has a malfunctioning internal battery. To stay powered, he must carry an external battery that slows him down and prevents him from jumping due to its weight.

This same battery is also used to activate devices that help him progress through the level. The challenge is to stay alive while solving puzzles and navigating each room.

Level design


Levels were designed using Tiled, which allowed me to export JSON map files and integrate them easily into Phaser.

Diegetic Interface


I aimed to make the game fully diegetic. The menu and tutorial are part of the game world itself. For example, I initially tried to display the battery level through a lightning icon on the robot’s back.

However, it wasn’t clear enough for players, so I added a more traditional UI element to display it.
