Maze is a game made without a game engine or graphics library; it was created entirely using ASCII, specifically with colored squares.

The goal is to progress through a randomly generated maze, climbing floors until reaching the exit — all while avoiding enemies.

Maze creation


The maze is generated using Kruskal's algorithm, which creates a simple maze (with only one possible path to the end).

To solve this problem, I randomly removed several walls in the maze, making it more complex. As you go up through the levels, their size decreases.

The game is fully modular, which allowed us to create difficulty modes by increasing the maze size, the number of enemies, and the number of floors.

We also added a “custom” mode where players can define these variables themselves to tailor the experience as they like.

They can even recreate the same maze using a seed system to control the randomness.

Saving and loading game


The player can save at any time while in the maze, which stores the current maze layout, collected items, their position, and the enemies’ positions. This way, they can resume exactly where they left off!

To do this, we write all the necessary data to a text file. When the player wants to continue, we simply read the file and recreate the state from that information.
