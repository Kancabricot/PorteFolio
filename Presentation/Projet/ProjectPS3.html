Outils de Travail

Pour réaliser ce projet, nous avons utilisé Unity pour le moteur de jeu, Blender pour la modélisation 3D, et Adobe Photoshop pour les textures. Le langage principal de développement est C# pour la logique du jeu et le scripting. Nous avons également utilisé Git pour le contrôle de version et Trello pour la gestion des tâches.

J'ai appris à travailler en équipe, à organiser mon code de manière efficace, et à optimiser les performances du jeu pour une meilleure expérience utilisateur. Ce projet a renforcé ma confiance en mes capacités de développement.
